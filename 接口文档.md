# 高校智慧校园门户系统 API 接口文档 v1.0

## 文档说明

- **Base URL**: `http://localhost:8081/api`
- **响应格式**: JSON
- **字符编码**: UTF-8
- **认证方式**: JWT Token（除登录注册外）

### 通用响应结构

```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {}
}
```



### 通用状态码

| code | 说明             |
| :--- | :--------------- |
| 200  | 成功             |
| 400  | 参数错误         |
| 401  | 未授权/Token过期 |
| 403  | 无权限           |
| 404  | 资源不存在       |
| 500  | 服务器错误       |

## 模块1：用户登录与统一认证

### 1.1 账号密码登录

**接口**: `POST /auth/login`

**请求参数**:

json

```json
{
  "username": "admin",      // 学号/工号/用户名
  "password": "123456"      // 密码
}
```



**响应示例**:

json

```json
{
  "code": 200,
  "msg": "登录成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiJ9...",
    "userId": 1,
    "username": "admin",
    "realName": "管理员",
    "userType": 3,          // 1-学生 2-教师 3-管理员
    "avatar": "https://...",
    "expiresIn": 7200000
  }
}
```
