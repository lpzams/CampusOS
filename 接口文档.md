# 高校智慧校园门户系统 API 接口文档 v3.0（完整版）

## 文档说明

- **Base URL**: `http://localhost:8081/api`
- **响应格式**: JSON
- **字符编码**: UTF-8
- **认证方式**: JWT Token（除登录注册外）

### 通用响应结构

json

```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {}
}
```



### 通用状态码

| code | 说明             |
| :--- | :--------------- |
| 200  | 成功             |
| 400  | 参数错误         |
| 401  | 未授权/Token过期 |
| 403  | 无权限           |
| 404  | 资源不存在       |
| 500  | 服务器错误       |

### 2.3 上传图片（新增）

**接口**: `POST /common/upload/image`

**请求头**: `Authorization: Bearer {token}`

**请求方式**: `multipart/form-data`

| 参数名 | 类型   | 必填 | 说明                                                         |
| :----- | :----- | :--- | :----------------------------------------------------------- |
| file   | File   | 是   | 图片文件（支持 jpg/png/webp，最大 5MB）                      |
| module | String | 否   | 业务模块标识，用于分类存储：`avatar`、`news`、`product`、`activity`、`repair`，默认 `common` |

**响应示例（成功）**:

json

```json
{
  "code": 200,
  "msg": "上传成功",
  "data": {
    "fileUrl": "https://cdn.example.com/uploads/20240615/avatar_abc123.jpg",
    "fileId": "file_20240615001",
    "fileSize": 102400,
    "fileName": "avatar_abc123.jpg"
  }
}
```



**响应示例（失败-文件过大）**:

json

```json
{
  "code": 400,
  "msg": "文件大小不能超过5MB",
  "data": null
}
```



**响应示例（失败-格式不支持）**:

json

```json
{
  "code": 400,
  "msg": "仅支持 jpg、png、webp 格式的图片",
  "data": null
}
```
