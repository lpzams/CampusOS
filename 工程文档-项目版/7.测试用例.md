## 1. 测试环境

- Windows 11；Node.js 21.7.1、npm 10.5.0。
- Docker Desktop：MySQL 8.0、Redis 7、Temurin JDK 17.0.19、CampusOS 后端及网站端。
- 服务地址：后端 `http://127.0.0.1:8080`，网站 `http://127.0.0.1:8081`。
- 宿主机 Maven 3.6.1 使用 JDK 11，不满足项目 Java 17 要求；领域规则改在项目 Docker JDK 17 环境执行。
