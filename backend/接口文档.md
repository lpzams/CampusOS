# 高校智慧校园门户系统 API 接口文档 v1.0（完整版）

## 文档说明

- **Base URL**: `http://localhost:8081/api`
- **响应格式**: JSON
- **字符编码**: UTF-8
- **认证方式**: JWT Token（除登录注册外）

### 通用响应结构

json

```
{
  "code": 200,
  "msg": "操作成功",
  "data": {}
}
```



### 通用状态码

| code | 说明             |
| :--- | :--------------- |
| 200  | 成功             |
| 400  | 参数错误         |
| 401  | 未授权/Token过期 |
| 403  | 无权限           |
| 404  | 资源不存在       |
| 500  | 服务器错误       |
