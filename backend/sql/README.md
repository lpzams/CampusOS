# SQL 脚本说明

本目录存放数据库 DDL（建表/改表）脚本，按版本号命名，方便追溯每次变更。

## 命名规范

```
V<主版本>.<次版本>__<描述>.sql
```

- `V1.0` — 初始建表
- `V1.1` — 第一次增量变更（加表/加字段/加索引）
- 以此类推

## 使用方式

```bash
# 连接 MySQL 后执行（按版本顺序）
mysql -u root -p campus < V1.0__init_user_table.sql
```
