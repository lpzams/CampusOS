# 高校智慧校园门户系统（Web + 小程序）15大功能模块

## 1. 用户登录与统一认证模块

**功能：**

- 学生/教师/管理员登录
- 手机验证码登录
- 学号/工号登录
- 微信授权登录
- 单点登录多个校园系统

支持端：

- 网站端
- 微信小程序端

------

# 2. 个人信息中心模块

**功能：**

- 查看个人资料
- 修改头像、联系方式
- 学籍信息查看
- 教师个人档案
- 实名认证

例如：

学生：

```

学院
专业
班级
学号
入学年份
```

教师：

```
姓名
学院
职称
工号
研究方向
```

```
             用户

        Web网站     微信小程序
             |
             |
        API网关(Spring Boot)
             |
 --------------------------------
 |       |       |       |
用户中心 新闻中心 教务中心 生活服务
 |
 MySQL + Redis
 |

```
